# xkcd-viewer

An alternate front-end for consuming the [xkcd](//xkcd.com) webcomic.

## Features

- Minimal
- Responsive
- Alt/title text below the image.
- Each comic links to its [explainxkcd](//explainxkcd.com) page.
- Each comic has a download link.
